# LNU AI Society Website

## Getting Started

Follow these steps to run the website locally:

1. **Clone the repository**

```bash
   git clone git@github.com:LNUAIS/Frontend.git
```

2. **Navigate to the project directory**

```bash
   cd Frontend
```

3. **Make sure you're on the dev branch**

```bash
   git checkout dev
```

4. **Start the local server**

```bash
   node scripts/server.js
```

5. **View the website on desktop**
   Open your browser on desktop and visit: [http://localhost:3000/](http://localhost:3000/)

---

6. **View the website on mobile**

   Turn off VPNs or Proxies on desktop and mobile phone.

   Check your Local IP address:
   - **Linux**: Open Terminal and run `ip addr` or `ifconfig`
   - **macOS**: Open Terminal and run `ifconfig` or check System Preferences → Network
   - **Windows**: Open Command Prompt and run `ipconfig`

   Make sure both your computer and phone are on the same WiFi network.

   Look for the field saying "IPv4 Address" (usually starts with `192.168.` or `10.0.`)

   Open your browser on mobile and visit: `http://YOUR_IP_ADDRESS:3000/`

   Example: `http://192.168.1.105:3000/`

## Configuration

The API base URL is configured in `scripts/config.js`. Default is `http://localhost:8080`.
